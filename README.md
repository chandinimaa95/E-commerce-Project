# E-commerce-Project
A Full-Stack ecommerce web application that provides seamless online shopping experience

🛒 E-Commerce Website

A Django-based E-Commerce web application developed using Python, Django, HTML, CSS, JavaScript, and SQLite.

📌 Features

- User Registration & Login
- Product Listing
- Product Details Page
- Shopping Cart
- Wishlist
- Search Products
- Place Order
- Remove from Cart
- Responsive Navigation Bar
- Admin Panel

  Live Demo:
  https://e-commerce-project-od0l.onrender.com/

🛠️ Technologies Used

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQLite
- Git & GitHub

🚀 Installation

1. Clone the repository.
2. Open the project folder.
3. Install dependencies:

pip install -r requirements.txt

4. Run migrations:

python manage.py migrate

5. Start the development server:

python manage.py runserver

6. Open your browser and visit:

http://127.0.0.1:8000/
